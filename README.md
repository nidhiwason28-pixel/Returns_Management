# Returns Management &amp; Refund Calculation Module
Summary

Designed and developed a complete Returns Management system for a U.S. market e-commerce platform operating on a consultant-driven sales model (similar to Amway). The work included building an end-to-end returns flow, refund calculation logic, and a frontend returns dashboard.

Architecture & Responsibilities

  Backend (Java, Apache OFBiz Framework)
  
    1. Implemented the returns workflow including validation, status transitions, and integration with order and inventory modules.
    2. Built the refund calculation engine considering promotions, taxes, stacked discounts, shipping charges, and return rules.
    3. Enhanced service layers and entity models for reliable return handling.
  Frontend (AngularJS, jQuery, HTML, CSS, JavaScript)
  
     1. Developed an interactive Returns Dashboard for consultants and customers to track request status, history, and eligibility.
     2. Implemented UI flows for initiating returns, viewing refund breakdown, and handling validations.
     3. Improved user experience with real-time updates and cleaner navigation.

Key Features

  1. Automated refund computation with accurate tax & discount adjustments
  2. Configurable return rules and product eligibility logic
  3. Intuitive dashboard with real-time status tracking
  4. Seamless integration with order, promotion, and payment subsystems

Tech Stack

  Java, Junit, Apache OFBiz, AngularJS, jQuery, JavaScript, HTML, CSS
